# IT1214-Student-Grade-System

## Project Purpose
This is a Java console program for a private institute. 
Teachers can store student ID, name, and marks digitally instead of paper.

## How to Run
1. Install Java JDK
2. Open terminal in this folder
3. Compile: javac Student.java GradeManagementSystem.java
4. Run: java GradeManagementSystem
5. Use menu 1 to 5 for operations

## Features
1. Add Student
2. Display All Students  
3. Search Student by ID
4. Calculate Average Marks
