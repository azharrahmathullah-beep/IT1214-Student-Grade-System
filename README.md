# IT1214-Student-Grade-System

## Project Purpose
This is a Java console program for a private institute. 
Teachers can store student ID, name, and marks digitally instead of paper.

## How to Run
1. Compile: javac Student.java GradeManagementSystem.java
2. Run: java GradeManagementSystem
3. Use menu 1 to 5 for operations

## Features
1. Add new student - ID, Name, Marks
2. Display all students 
3. Search student by ID
4. Calculate average marks of all students
